# veri-filtreleme-siralama-projesi

Bu proje, verilen country veri seti kullanılarak temel veri analizi, filtreleme ve sıralama işlemlerini içermektedir.

## Kullanılan Teknolojiler
- Python
- Pandas
- Jupyter Notebook

## Yapılan Görevler
- Nüfusa göre ülkeleri azalan sırada sıralama
- GDP per capita değerine göre sıralama
- Belirli nüfus ve GDP kriterlerine göre ülke filtreleme
- Okuryazarlık oranı ve nüfus yoğunluğu analizleri

Bu çalışma Techcareer.net Veri Bilimine Giriş bitirme projesi kapsamında hazırlanmıştır.
